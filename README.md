# Christ Embassy Automation

This project automates weekly devotional generation and dispatch via email and SMS. It also provides a Streamlit dashboard for managing topics.

## Features

- AI-generated weekly devotionals using OpenAI
- Email sending via SendGrid
- SMS notifications via Twilio
- Streamlit dashboard

## Usage

1. Configure your `.env` with API keys.
2. Run `streamlit run app.py` to start the dashboard.
3. Use the UI to generate and send devotionals.

## Deployment

Deploy easily on [Streamlit Community Cloud](https://streamlit.io/cloud).
